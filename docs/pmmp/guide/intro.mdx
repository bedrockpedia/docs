## What is PocketMine-MP?

[PocketMine-MP](https://github.com/pmmp/PocketMine-MP) is a custom server software for Minecraft: Bedrock Edition. It is written in PHP and has been around for a long time, serving as the original custom server software for MCPE.

It allows you to host your own server with a powerful plugin API to customize your gameplay.

---

## Why Use PocketMine-MP?

PocketMine-MP is arguably the most famous and widely-used custom server software for Minecraft: Bedrock Edition. When deciding whether to use it, consider its unique strengths and limitations.

- **Massive Community & Plugins**
  <div className="text-default-600 dark:text-default-500">
    Because it has been around since the very early days of Minecraft PE, PocketMine-MP boasts the largest plugin ecosystem of any Bedrock server software. 
  </div>

- **PHP-Based Architecture**
  <div className="text-default-600 dark:text-default-500">
    PocketMine is written entirely in PHP. This makes it incredibly easy and accessible for beginners to learn and start writing their own plugins without needing to compile code.
  </div>

- **Extreme Performance**
  <div className="text-default-600 dark:text-default-500">
    Designed for high-concurrency, PocketMine-MP can handle hundreds of players on a single server instance with virtually no lag, making it the top choice for massive network lobbies and minigames.
  </div>

- **Lack of Vanilla Parity (The Catch)**
  <div className="text-default-600 dark:text-default-500">
    PocketMine-MP does **not** feature vanilla mobs, animals, or redstone mechanics natively. While plugins can simulate these, they are never 1:1 with vanilla. Therefore, PocketMine-MP is **not recommended for standard Survival (SMP) servers**.
  </div>

- **Active Development**
  <div className="text-default-600 dark:text-default-500">
    The project is meticulously maintained by a dedicated team of developers who swiftly update the software alongside new Minecraft versions.
  </div>
