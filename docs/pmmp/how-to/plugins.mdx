# Plugins

Plugins extend your server's functionality by adding new features, commands, and gameplay mechanics.

## Finding Plugins

- **Poggit (Official)**: [poggit.pmmp.io/plugins](https://poggit.pmmp.io/plugins)

---

## Installing Plugins

<Steps>

### Download the Plugin
Download the `.phar` file from a trusted source like Poggit.

### Stop Your Server
Type `stop` in the console to safely shut down the server.

### Place the Plugin
Move the `.phar` file into the `plugins/` folder in your server directory.

### Start the Server
Launch your server. The plugin will load automatically.

### Verify Installation
Type `plugins` in the console to see a list of loaded plugins.

</Steps>

---

## Managing Plugins

### Listing Plugins
To see all active plugins:
```bash
plugins
```

### Plugin Configuration
Most plugins create their own folder in `plugins/` with a `config.yml` file:
```text
plugins/
├── MyPlugin.phar
└── MyPlugin/
    └── config.yml
```

Edit the `config.yml` (or similar file in `plugin_data/`) to customize plugin settings, then restart.

### Removing Plugins
1. Stop the server
2. Delete the plugin `.phar` file from `plugins/`
3. Optionally, delete the plugin's config folder inside `plugin_data/`
4. Restart the server

## Troubleshooting

<Blockquote color="danger">
  **Plugin Not Loading?** Check the `logs/server.log` file for error messages. Common issues include:
  - Incompatible plugin version
  - Missing dependencies
  - Conflicting plugins
</Blockquote>

### Common Fixes
- **Update the plugin**: Check for newer versions compatible with your server
- **Check dependencies**: Some plugins require other plugins to work
- **Read documentation**: Always check the plugin's installation guide
